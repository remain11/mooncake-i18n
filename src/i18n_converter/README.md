# i18n_converter


## Introduction
The pakage contains a function can convert a string to the form of "i18n" ,it can help compress space!

## Import the package

Add the package to your project, and import it in your code.

```
{
    "path": "{your_path}",
    "alias": "{your_alias}"
}
```

## Usage

```moonbit

let a = @{your_alias}.i18n("hello")

```
